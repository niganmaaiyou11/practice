# 通用 UI 设计规范

---

## 一、前端设计风格

### 1.1 Apple 风格设计语言

整体视觉体系以 Apple 设计语言为基准：

- **背景色**：浅灰 `#f5f5f7`
- **卡片圆角**：18-20px 大圆角
- **阴影**：微阴影层级（低饱和度、柔和扩散）
- **字体排版**：SF 风格字体（San Francisco 系统字体栈）

### 1.2 iOS 26 液态玻璃导航

液态玻璃质感导航栏，透明玻璃背景仅由镜面高光边缘和浮动阴影定义形态：

- **透明玻璃质感**：`background: rgba(255,255,255,0.1)`
- **镜面高光边缘**：`inset 0 0.5px` 细线定义玻璃边界
- **浮动阴影**：阴影定义玻璃形态，替代实色背景
- **液态光斑**：`requestAnimationFrame` + lerp 缓动，光斑跟随鼠标自由滑动
- **字符级折射**：光斑经过文字时逐字变色，CSS `transition 400ms` 平滑过渡
- **零依赖**：不依赖任何第三方动画库

### 1.3 玻璃拟态（Glassmorphism）

- **粘性头部**：`backdrop-filter: saturate(180%) blur(20px)`，滚动时呈现毛玻璃效果
- **悬浮提示**：毛玻璃 tooltip
- **导航栏**：液态玻璃 + 毛玻璃双层质感

### 1.4 CSS 自定义属性（设计令牌）

完整的设计令牌系统，通过 CSS 自定义属性统一管理：

- 颜色令牌
- 字体令牌
- 间距令牌
- 圆角令牌
- 阴影令牌
- 过渡动画令牌

## 二、交互动画设计风格

### 2.1 数字滚动动画

- **核心技术**：`requestAnimationFrame` + `easeOutCubic` 缓动函数
- **效果**：数字从 0 平滑滚动到目标值
- **应用场景**：统计卡片、数据指标展示

### 2.2 卡片悬浮抬升

- **触发方式**：鼠标 hover
- **效果**：卡片沿 Y 轴抬升 + 阴影加深
- **应用场景**：统计卡片、预览卡片、快捷入口卡片

### 2.3 页面渐入过渡

- **效果**：页面切换时的淡入过渡动画
- **方式**：路由级过渡动画

### 2.4 图表数据切换动画

- **图表类型切换**：面积图 / 折线图 / 柱状图之间切换时带过渡动画
- **数据更新**：筛选条件变化时图表系列数据带动画过渡

### 2.5 流动公告卡片轮播

- **实现方式**：CSS `@keyframes translateX` + 重复列表技法
- **滚动方式**：无缝水平循环滚动
- **悬浮暂停**：鼠标 hover 时动画暂停
- **边缘渐变遮罩**：左右边缘使用渐变透明遮罩

### 2.6 液态光斑 + 字符折射

- **光斑运动**：rAF 驱动，lerp 缓动插值实现平滑跟随
- **折射效果**：光斑经过导航文字时逐字变色
- **过渡速度**：CSS `transition 400ms`，柔和自然
- **交互方式**：鼠标在导航栏内移动触发

### 2.7 卡片悬浮位移动画

- **效果**：卡片悬浮时水平方向微微位移
- **应用场景**：排行榜预览卡片、列表项 hover

### 2.8 图表无限级缩放

- **手势支持**：鼠标滚轮缩放 + 底部滑动条
- **特性**：无极缩放 + 平移
- **触发条件**：数据项超过一定数量时自动启用

## 三、UI 设计风格

### 3.1 色彩系统

- **页面背景**：`#f5f5f7`（Apple 浅灰）

### 3.2 排版系统

- **字体栈**：SF 风格（`-apple-system, BlinkMacSystemFont, "Segoe UI", ...`）
- **标题层级**：大标题 / 页面标题 / 卡片标题 / 正文
- **数字展示**：大号数字使用等宽数字特性

### 3.3 毛玻璃效果规范

- **导航栏**：`backdrop-filter: blur(20px)` + `saturate(180%)`
- **Tooltip**：`backdrop-filter: blur(20px)` + 半透明背景
- **粘性头部**：滚动触发毛玻璃

### 3.4 多语言 UI

支持多语言切换，所有 UI 文字、组件库、图表标签同步切换：

- 语言选择持久化到本地存储
- 切换时 UI 文字、组件和图表标签同步更新

### 3.5 组件库覆写

基于第三方 UI 组件库，通过全局样式覆写以匹配设计语言：

- 自定义导航替代默认菜单组件
- 按钮、输入框、表格、分页等组件样式统一覆写

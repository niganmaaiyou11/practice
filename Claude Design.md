# Claude Design 

## 一、整体视觉风格与核心理念

Claude Design 文档网页主打**知性安静、简约文艺**阅读风，区别于科技风冷硬质感，偏向纸质书籍舒适阅读体验，整体低饱和度、无花哨装饰，侧重长文本沉浸式阅读。

- 设计理念：复刻纸质书籍阅读质感，简约稳重、专业儒雅

- 整体调性：温润素雅、极简留白、学术正式、商务高级

- 视觉特点：无夸张渐变、无炫酷动效、排版规整、层级清晰

## 三、字体系统（核心重点）

### 3\.1 全局字体优先级

1. 标题字体：**Anthropic Serif** 人文衬线体，备用 `Georgia`

2. 正文字体：**Anthropic Sans** 简约无衬线体，备用 `Inter / system\-ui`

3. 代码字体：**Anthropic Mono** 温润等宽字体，备用 `Consolas`

### 3\.2 字体使用规则

- 所有**大标题、章节标题、小标题**统一使用 **Anthropic Serif 衬线字体**，凸显正式学术感

- 所有**正文内容、导航栏、按钮、侧边栏**统一使用 **Anthropic Sans 无衬线字体**，保证易读性

- 代码片段、命令行、参数字段使用 **Anthropic Mono 等宽字体**

- 全篇极少使用粗体强调，多用**斜体**突出关键词，整体字体气质柔和统一

### 3\.4 字体排版核心规范

1. 全篇标题**统一字重 500**，不做超粗黑体，风格统一协调

2. 正文行高统一偏宽松（1\.6），接近纸质书籍排版，长时间阅读不累眼

3. 标题行高收紧，紧凑大气，留白克制

4. 小号字体轻微拉开字间距，提升小字清晰度

5. 英文长文本自动均匀断行，无拥挤挤压感

6. 中英文混排高度适配，衬线 \+ 无衬线搭配自然不冲突

## 四、页面布局样式

1. 整体居中布局，内容最大宽度 1200px，左右留白充足

2. 经典**左侧固定导航目录 \+ 右侧正文阅读区**文档结构

3. 间距以 8px 为基础单位，段落间距 16\-24px，章节大间距 80\-120px

4. 全篇大量留白，节奏舒缓，阅读层次分明

5. 内容区块统一圆角柔和设计，无尖锐直角

## 五、常用组件样式

- 代码块：浅象牙底色、细浅色边框、柔和圆角，左侧细强调色标识

- 提示信息框：浅底色 \+ 顶部细色条极简样式

- 表格：极简横线分隔，无复杂底色，表头使用衬线字体

- 按钮：低圆角、无厚重阴影，仅细微边框层次

- 滚动高亮：侧边目录跟随阅读位置自动高亮对应标题

## 六、动效与交互风格

1. 所有过渡动画时长控制 200ms 以内，极简克制

2. 链接、按钮仅做底色微妙变化，无缩放、无晃动

3. 支持深色模式，冷暖色调同步适配，字体风格完全不变

4. 全程平滑滚动，无突兀视觉跳转
